# This is NOT the Next.js you know

This repo runs Next.js 16 (App Router, Turbopack) with React 19. APIs, conventions,
and file structure differ from older versions you may have learned. Read the relevant
guide in `node_modules/next/dist/docs/` before writing or changing any framework code,
and heed deprecation notices. Route `params`/`cookies`/`headers` are async.
